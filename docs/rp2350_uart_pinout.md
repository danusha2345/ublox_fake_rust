## Оптимизация для компактной PCB

### Вариант 1: Всё на левой стороне (пины 2-9) — РЕКОМЕНДУЕТСЯ

```
Пин  GPIO  Функция
───────────────────
 2   GPIO0   UART0 TX (к дрону)
 3   GPIO1   UART0 RX (от дрона)
 4   GPIO2   Кнопка PWR
 5   GPIO3   Кнопка INPUT
 6   DVDD    (питание)
 7   GPIO4   UART1 TX (резерв)
 8   GPIO5   UART1 RX (от GNSS)
 9   GPIO6   LED WS2812 (опционально)
```

**Плюсы:** Все сигналы на 7 соседних пинах одной стороны. Минимальная разводка.

### Вариант 2: Нижняя сторона (пины 16-27)

```
Пин  GPIO   Функция
────────────────────
16   GPIO12  UART0 TX
17   GPIO13  UART0 RX
18   GPIO14  Кнопка PWR
19   GPIO15  Кнопка INPUT
27   GPIO16  LED WS2812
```

UART1 RX: GPIO5 (пин 8, левая сторона) или GPIO21 (пин 33, правая сторона)

### Вариант 3: Правая сторона (пины 32-37)

```
Пин  GPIO   Функция
────────────────────
32   GPIO20  UART1 TX (резерв)
33   GPIO21  UART1 RX (от GNSS)
34   GPIO22  Кнопка PWR
35   GPIO23  Кнопка INPUT
36   GPIO24  LED
37   GPIO25  (резерв)
```

UART0: GPIO28/29 (пины 42/43) — ADC сторона
